🚦 Nexus Traffic AI

An AI-powered Smart Traffic Management System designed to optimize traffic flow, reduce congestion, and prioritize emergency vehicles using real-time data and predictive analytics.

---

🌟 Features

* 🚦 **Smart Signal Control**
  Dynamic traffic light management based on real-time vehicle load

* 🗺️ **Live Intelligence Map**
  Real-time visualization of traffic conditions at intersections

* 🚑 **Emergency Green Corridor (USP)**
  Automatically creates a green path for ambulances, fire trucks, and police vehicles

* 📊 **Deep AI Analytics**
  Time-series tracking and machine learning-based traffic prediction

* 📡 **Live Node Telemetry**
  Real-time data from traffic nodes (vehicle count, density, status)

* ⚙️ **Signal Control Panel**
  Switch between AI-controlled mode and manual override

🧠 How It Works

1. Cameras capture real-time traffic data
2. AI detects and counts vehicles
3. Backend processes data and calculates traffic load
4. Signals are dynamically controlled
5. Emergency vehicles trigger green corridors
6. Analytics predict future congestion
